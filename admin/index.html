Cara Setup Google Sheets RSVP

1
Buat Google Sheet baru

Buka sheets.google.com → buat spreadsheet baru → beri nama "RSVP Undangan".

2
Buka Script Editor

Klik menu Extensions → Apps Script. Hapus kode yang ada, lalu paste kode dari file admin/apps-script.js.

3
Deploy sebagai Web App

Klik Deploy → New deployment → pilih type Web app. Set Execute as: Me dan Who has access: Anyone. Klik Deploy.

4
Salin URL Web App

Salin URL yang diberikan (berbentuk https://script.google.com/macros/s/.../exec), lalu paste di field "URL Google Apps Script" di form editor.

5
Selesai!

RSVP dari tamu akan otomatis masuk ke Google Sheets Anda. Cek di halaman RSVP Dashboard.

Kode Apps Script: 
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    sheet.appendRow([
      new Date(), data.invId, data.couple,
      data.name, data.attendance, data.message
    ]);
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSheet();
  var rows = sheet.getDataRange().getValues();
  var result = [];
  for (var i = 1; i < rows.length; i++) {
    result.push({
      timestamp: rows[i][0],
      invId: rows[i][1],
      couple: rows[i][2],
      name: rows[i][3],
      attendance: rows[i][4],
      message: rows[i][5]
    });
  }
  return ContentService
    .createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}

Panduan Penggunaan

1
Live Editor

Buka halaman Live Editor. Isi semua data klien di form kiri. Preview undangan akan terupdate otomatis di sebelah kanan secara real-time.

2
Our Story

Isi bagian "Perjalanan Cinta" untuk menambahkan timeline kisah cinta mempelai di undangan (minimal 2 momen).

3
Google Sheets RSVP

Setup Google Apps Script sekali saja, lalu masukkan URL-nya. RSVP tamu akan masuk otomatis ke spreadsheet Anda.

4
Generate & Simpan

Setelah selesai, klik Generate Link untuk membuat tautan. Klik Simpan untuk menyimpan ke daftar undangan.

5
Daftar Tamu

Buka halaman Daftar Tamu. Pilih undangan yang sudah dibuat, lalu paste daftar nama tamu (satu per baris). Sistem akan generate link unik per tamu.

6
Share via WhatsApp

Di tabel Daftar Tamu, klik tombol WhatsApp di samping setiap nama untuk langsung membuka chat WA dengan pesan dan link yang sudah terisi otomatis.

Cara Setup Appwrite Cloud

1
Buat Akun Appwrite & Project

Daftar di cloud.appwrite.io, dan klik Create Project.

2
Dapatkan Project ID & Endpoint

Ke menu Settings. Salin **API Endpoint** (biasanya https://cloud.appwrite.io/v1) dan **Project ID** ke form.

3
Buat Database

Masuk ke menu Databases -> Create Database (Contoh nama: invites_db). Salin nilain Database ID.

4
Buat Collection

Di dalam database yang baru dibuat, klik Create Collection (Contoh nama: saved_invites). Salin Collection ID.

5
Buat Attributes

Masuk ke koleksi, ke tab Attributes dan buat field tipe String berikut:
- uid (size 50)
- groom (size 100)
- bride (size 100)
- date (size 100)
- template (size 50)
- url (size 2000)
- formData (size 5000)

6
Update Permissions (PENTING)

Pergi ke tab Settings dalam Collection. Pada bagian Permissions, centang Any, lalu pilih akses Read, Create, Update, Delete agar admin panel bisa menyimpan dan membaca data.
